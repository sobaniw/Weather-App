# Weather Web App 🌤️

A beautiful weather application built with HTML, CSS, and JavaScript.

## Features

- 🔍 Search weather by city name
- 📊 Current weather display (temperature, humidity, wind speed, pressure, visibility, UV index)
- 📅 7-day forecast
- ⏰ Hourly weather breakdown (click any day to see hourly details)
- 📱 Fully responsive design
- 🎨 Beautiful UI with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name in the search box
3. Click the Search button or press Enter
4. View current weather and 7-day forecast
5. Click on any day to see 24-hour hourly forecast

## Files

- `index.html` - HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Weather app logic and API calls

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Open-Meteo API (Free weather data, no API key required)

## API Information

This app uses the free [Open-Meteo API](https://open-meteo.com/) which provides:
- Current weather data
- 7-day forecast
- Hourly weather data
- Geocoding (city to coordinates)

### Current Weather

- Hourly temperature

## ✨ Features

### 🔍 **City Search**
- Search weather by city name worldwide
- Real-time geocoding using Open-Meteo API
- Instant results with error handling
- Works with any city globally

### 🌡️ **Current Weather Display**
- Real-time temperature in Celsius
- Live weather conditions (sunny, cloudy, rainy, etc.)
- Detailed weather metrics:
  - **Feels Like Temperature** - Actual feel vs actual temp
  - **Humidity** - Moisture level in air (%)
  - **Wind Speed** - Current wind speed (km/h)
  - **Pressure** - Atmospheric pressure (hPa)
  - **Visibility** - How far you can see (km)
  - **UV Index** - Sun intensity with color-coded risk level

### 📅 **7-Day Forecast**
- Daily high and low temperatures
- Weather icons for quick visual reference
- Weather condition descriptions
- Click any day to see hourly details
- All 7 days visible at once

### ⏰ **Hourly Weather Breakdown**
- 24-hour forecast for selected day
- Hourly temperature
- Weather conditions for each hour
- Precipitation probability
- Wind speed per hour
- Smooth horizontal scrolling
- Easy close button to return to daily view
