# Weather Web App 🌤️

A beautiful weather application built with HTML, CSS, and JavaScript.

## Features

- 🔍 Search weather by city name
- 📊 Current weather display (temperature, humidity, wind speed, pressure, visibility, UV index)
- 📅 7-day forecast
- ⏰ Hourly weather breakdown (click any day to see hourly details)
- 📱 Fully responsive design
- 🎨 Beautiful UI with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name in the search box
3. Click the Search button or press Enter
4. View current weather and 7-day forecast
5. Click on any day to see 24-hour hourly forecast

## Files

- `index.html` - HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Weather app logic and API calls

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Open-Meteo API (Free weather data, no API key required)

## API Information

This app uses the free [Open-Meteo API](https://open-meteo.com/) which provides:
- Current weather data
- 7-day forecast
- Hourly weather data
- Geocoding (city to coordinates)

## Features Explained

### Current Weather
Shows real-time temperature, weather condition, and detailed metrics:
- Feels like temperature
- Humidity
- Wind speed
- Pressure
- Visibility
- UV Index (with risk level assessment)

### 7-Day Forecast
View weather for the next 7 days with:
- High/Low temperatures
- Weather conditions
- Weather icons

### Hourly Forecast
Click any day to see hourly breakdown for that day with:
- Hourly temperature
- Weather condition
- Precipitation probability
- Wind speed
