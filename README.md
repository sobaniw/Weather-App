# 🌤️ Weather Web App

A modern and responsive Weather Web Application built using HTML, CSS, and JavaScript.  
This app provides real-time weather data, a 7-day forecast, and an interactive hourly breakdown with a clean and user-friendly UI.

## ✨ Features

- 🔍 Search weather by city name worldwide  
- 🌡️ Real-time weather details (temperature, humidity, wind, pressure, visibility, UV index)  
- 📅 7-day weather forecast  
- ⏰ Interactive hourly weather breakdown  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🎨 Modern UI with glassmorphism effects and smooth animations  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Animations, Glassmorphism UI)  
- JavaScript (ES6+)  
- Open-Meteo API (No API key required)  

---

## ⚙️ How It Works

1. User enters a city name  
2. The app converts the city into geographic coordinates (geocoding)  
3. Weather data is fetched from the Open-Meteo API  
4. Data is dynamically displayed on the UI
